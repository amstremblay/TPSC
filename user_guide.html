TPSC

TPSC calculation are handled by the TPSC class.

class TPSC.TPSC(n, U, t, tp, tpp, nkx, dispersion_scheme, T, wmax_mult=10, IR_tol=1e-12)

Class to set up a TPSC calculation. Calculation is carried using the run() method.

Parameters:

  • n (double) – Density

  • U (double) – Hubbard interaction

  • t (double) – First neighbour hopping

  • tp (double) – Second neighbour hopping

  • tpp (double) – Third neighbour hopping

  • nkx (int) – Number of k-points in one space direction

  • dispersion_scheme (str) – Dispersion scheme (either triangle or square)

  • T (double) – Temperature

  • wmax_mult (double, optional) – For IR basis, multiple of bandwidth to use as wmax (must be greater than 1)

  • IR_tol (double, optional) – For IR basis, tolerance of intermediate representation (default = 1e-12)

plotSelfEnergyVsWn(coordinates, Wn_range, show=True, ax=None)

Evaluate and plot the self-energy as a function of Wn for a specific k-point.

Parameters:

  • coordinates (list) – The k-point to evaluate the self-energy

  • Wn_range (list) – x-axis range

  • show (bool, optional) – Whether or not showing the grah

  • ax (optional) – A matplotlib axis to plot the grah (default, create a graph)

Returns:

A matplotlib axis objet containing the graph
